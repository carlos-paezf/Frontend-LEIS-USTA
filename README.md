## Comandos

- Crear un proyecto con Angular CLI 13
  
  ```txt
  ng new frontend-leis
  ```

- Reinstalar o recuperar los `node_modules`
  
  ```txt
  npm install
  ```

- Desplegar el Servidor y abrirlo en el navegador
  
  ```txt
  ng serve -o
  ```
